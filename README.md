# Movie Library (Личная кинотека)

Автор: Акулов Никита

## Описание

Movie Library — это настольное приложение с графическим интерфейсом, разработанное на Python (Tkinter), предназначенное для хранения, поиска и управления списком фильмов.

Программа позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Также реализованы функции поиска, фильтрации и удаления записей. Все данные сохраняются в JSON-файл и автоматически загружаются при запуске.

## Возможности

- добавление фильмов
- удаление фильмов
- поиск по названию
- фильтрация по жанру
- фильтрация по году выпуска
- хранение данных в JSON
- автоматическая загрузка данных

## Проверка ввода

- год должен быть числом
- рейтинг должен быть от 0 до 10
- название не может быть пустым
